# 🐦 Twitter Clone

Spring Boot와 React로 구현한 트위터 클론 프로젝트입니다.  
JWT 로그인, GitHub 소셜 로그인(OAuth2), 게시글 CRUD, 댓글, 좋아요, 팔로우 기능을 포함합니다.

---

## ⚙️ Tech Stack
**Backend:** Java 17 · Spring Boot · Spring Security · JPA · H2 · JWT  
**Frontend:** React · Vite · TailwindCSS · Axios
